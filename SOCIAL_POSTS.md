## LinkedIn Post

🧙‍♂️ Apresento o Nikito's Ledger — uma ferramenta open source para gerar fichas de D&D 5.5 automaticamente com IA.

A ideia nasceu de uma frustração simples: jogadores e mestres perdem tempo demais preenchendo formulários e gerenciando micro-detalhes de fichas, quando o que realmente importa é a história.

Com o Nikito's Ledger, você descreve seu personagem em linguagem natural — "Elfa Ranger nível 5, guardiã de uma floresta amaldiçoada" — e recebe uma ficha completa com stats, habilidades, magias e backstory. Tudo calculado pelas regras oficiais do D&D 5.5.

Para mestres, há um gerador de NPCs que cria personagens memoráveis com notas de interpretação, motivações secretas e ganchos de aventura — tudo que um DM precisa para improvisar na mesa.

Principais features:
• Prompt-to-Sheet via Gemini AI
• Gerador de NPCs com notas exclusivas para o mestre
• Importação de Markdown (para quem já criou personagens em outras IAs)
• Suporte a imagem como retrato do personagem
• Identidade visual única por personagem
• Exportação para PDF estilizado

Stack técnica: Vue 3, Pinia, Tailwind CSS 4, Google Gemini Flash 2.5, pdf-lib.

UI inspirada em Baldur's Gate 3 com estética de pergaminho dos livros clássicos de Forgotten Realms.

O projeto é open source e feito pela comunidade, para a comunidade.
